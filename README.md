JD首页（部分）

一：项目描述

	这是学习完初级的HTML5+CSS3时做的一个简单的京东一部分首页

	作为练习使用

	项目让我第一次学习到了公共初始化CSS

	以及如何使用阿里矢量图标，第一次认识到了图标字体这个概念，感受到了图标字体的强大

	还学习到了如何下载网站图标和设置网站图标
